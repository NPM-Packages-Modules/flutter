# SecureVaultLite

Complete secure local storage ecosystem.

## Features

- Encrypted storage
- Secure file vault
- PIN/biometric lock
- Auto session expiration
- Secret sharing between isolates

## Usage

```dart
import 'package:secure_vault_lite/secure_vault_lite.dart';

void main() {
  SecureVaultLite.init();
}
```

Most apps implement security poorly.
