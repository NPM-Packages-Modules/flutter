# flutter-device-lab

Run automated testing across device matrices.

## CLI

```bash
npx flutter-device-lab run
```

## Features

- emulator spawning
- screenshots
- logs
- reports

## Usage

```dart
import 'package:flutter_device_lab/flutter_device_lab.dart';

void main() {
  FlutterDeviceLab.init();
}
```

## Why

Cross-device testing setup is painful.
