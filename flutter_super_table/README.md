# FlutterSuperTable

**Topics:** `dart` · `data-grid` · `flutter` · `flutter-super-table` · `mobile` · `pub` · `table`

Enterprise-level data table package.

## Features

- Virtual scrolling
- Column resize
- Filters
- Inline editing
- Frozen columns
- Excel-like keyboard navigation
- CSV/Excel export

## Usage

```dart
import 'package:flutter_super_table/flutter_super_table.dart';

void main() {
  FlutterSuperTable.init();
}
```

## Why

Flutter lacks truly powerful data table solutions.

**High market potential** — production pain point.
