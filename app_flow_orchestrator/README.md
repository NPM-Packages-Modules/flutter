# AppFlowOrchestrator

Visual screen flow/state orchestration engine.

## Features

- Navigation graph builder
- Conditional routing
- Auth guards
- Deep-link manager
- Feature flags
- Screen analytics

## Usage

```dart
import 'package:app_flow_orchestrator/app_flow_orchestrator.dart';

void main() {
  AppFlowOrchestrator.init();
}
```

## Why

Navigation complexity becomes huge in large apps.
