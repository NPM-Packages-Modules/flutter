# SmartFormX

AI-powered form engine for Flutter.

## Features

- Auto validation
- Dynamic forms from JSON
- Multi-step wizard
- Auto-save drafts
- Conditional fields
- Built-in OTP inputs
- File upload widgets
- Async validators

## Usage

```dart
import 'package:smart_form_x/smart_form_x.dart';

void main() {
  SmartFormX.init();
}
```

## Why

Developers waste huge time building forms repeatedly.

**High market potential** — production pain point.
