# flutter-firelink

**Topics:** `dart` · `flutter` · `flutter-firelink` · `mobile` · `pub`

Firebase setup in one command.

## CLI

```bash
npx flutter-firelink connect
```

## Features

- Firebase apps
- config files
- auth setup
- analytics
- crashlytics

## Usage

```dart
import 'package:flutter_firelink/flutter_firelink.dart';

void main() {
  FlutterFirelink.init();
}
```

## Why

Firebase integration still involves many steps.
