# flutter-zero-setup

**Topics:** `cli` · `dart` · `flutter` · `flutter-zero-setup` · `mobile` · `pub` · `scaffolding`

Initialize a production-grade Flutter app in one command.

## CLI

```bash
npx flutter-zero-setup init
```

## Features

- flavors
- Firebase
- CI/CD
- folder architecture
- env handling
- linting
- localization

## Usage

```dart
import 'package:flutter_zero_setup/flutter_zero_setup.dart';

void main() {
  FlutterZeroSetup.init();
}
```

## Why

Flutter create gives bare minimum setup.
