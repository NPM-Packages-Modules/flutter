# flutter-state-smith

Generate complete state management architecture.

## CLI

```bash
npx flutter-state-smith create auth
```

## Features

- state
- providers
- repositories
- usecases
- tests

## Usage

```dart
import 'package:flutter_state_smith/flutter_state_smith.dart';

void main() {
  FlutterStateSmith.init();
}
```

## Why

Riverpod/BLoC setup is repetitive.
