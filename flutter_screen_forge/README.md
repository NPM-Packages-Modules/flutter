# flutter-screen-forge

Generate complete screens from prompts.

## CLI

```bash
npx flutter-screen-forge create "product listing page"
```

## Features

- loading states
- pagination
- filters
- shimmer

## Usage

```dart
import 'package:flutter_screen_forge/flutter_screen_forge.dart';

void main() {
  FlutterScreenForge.init();
}
```

## Why

Creating repetitive CRUD screens wastes hours.
