# flutter-route-genius

Auto-generate type-safe navigation.

## CLI

```bash
npx flutter-route-genius scan
```

## Features

- Type-safe routes
- Deep links
- Route analytics

## Usage

```dart
import 'package:flutter_route_genius/flutter_route_genius.dart';

void main() {
  FlutterRouteGenius.init();
}
```

## Why

Navigation boilerplate is huge.
