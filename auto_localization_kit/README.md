# AutoLocalizationKit

**Topics:** `auto-localization-kit` · `dart` · `flutter` · `i18n` · `localization` · `mobile` · `pub`

Fully automated localization system.

## Features

- AI translation integration
- Auto key extraction
- Missing translation detector
- RTL auto-fixes
- Locale-aware formatting

## Usage

```dart
import 'package:auto_localization_kit/auto_localization_kit.dart';

void main() {
  AutoLocalizationKit.init();
}
```

## Why

Localization setup is repetitive and messy.
