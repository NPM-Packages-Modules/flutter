# MotionBuilder

No-code animation system for Flutter.

## Features

- Timeline animations
- Drag-drop animation configs
- Gesture-based transitions
- Preset effects
- Shared element transitions

## Usage

```dart
import 'package:motion_builder/motion_builder.dart';

void main() {
  MotionBuilder.init();
}
```

## Why

Animations in Flutter are powerful but verbose.
